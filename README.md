# Ethiopia Clinic Appointment & Queue System

A full-stack web application to reduce long waiting times in Ethiopian clinics by providing appointment booking and live queue tracking.

## Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT

## Features

- Patient registration & login
- Book appointments by department
- Live queue position
- Clinic dashboard for managing patients

## Real-World Problem

Many clinics still use paper queues. Patients wait hours without knowing their turn.  
This system improves efficiency and patient experience.
